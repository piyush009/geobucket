# GeoBucket

Decision science, AI/ML, blockchain, and infrastructure software consultancy.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Docker

```bash
docker compose up --build
```

Or without Compose:

```bash
docker build -t geobucket .
docker run --rm -p 3000:3000 geobucket
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

Edit site copy in `content/`.
